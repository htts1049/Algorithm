### 주의사항 
- 문자열을 정수로 치환할 때 문자열 인덱스가 문자열 길이가 넘어가는 지 확인 필요
- 벡터 비우기 주의
- 테스트 케이스 * 함수 개수 * 정수 배열 길이 = 100 * 100,000 * 100,000 = 10^12 이므로, 정수 배열에 접근하는 시간 복잡도를 O(1) 로 줄여야 함

### 풀이
- 파싱 + 덱 문제
- 문자열에 담긴 숫자 값들을 정수로 치환한 뒤 정방향이면 front를, 역방향이면 rear를 pop
